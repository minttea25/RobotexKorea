# RobotexKorea
Lottery Program (Formation, Ticket) for Robotex Korea 2022

# Excel Sheets
The selectable excel file should have sheets:
- Robotex
- LegoSumo1kg
- LegoSumo3kg
- LineFollowingE
- LineFollowingJH
- LegoFolkraceE
- LegoFolkraceJH
- RoboLeague

# Valid Rows
If the row has a value on column 'No.', it is valid data.

1.8
